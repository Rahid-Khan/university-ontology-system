# University Management Ontology System

A comprehensive desktop application for managing university knowledge graphs using semantic web technologies.

## Features

- **Ontology Management**: Create, edit, and visualize university ontology
- **SPARQL Query Interface**: Advanced query editor with templates and history
- **Interactive Visualizations**: Dynamic graphs with zoom, pan, and filtering
- **Data Import/Export**: Multiple formats including Turtle, RDF/XML, JSON-LD, CSV
- **Sample Data**: Pre-loaded university dataset for quick start
- **Professional GUI**: Modern interface with tabs, toolbars, and status bars
